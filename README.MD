# Programación II — TUP (C# / .NET)

Este repositorio contiene todo el material, ejercicios y proyectos desarrollados durante la cursada de **Programación II** de la **Tecnicatura Universitaria en Programación (TUP)**.

El enfoque principal del cuatrimestre es la **Programación Orientada a Objetos (POO) usando C# y .NET**.


## 🎯 Objetivos de la materia

- Comprender los principios fundamentales de la **POO**  
- Aplicar conceptos como **clases**, **objetos**, **encapsulamiento**, **herencia**, **polimorfismo**  
- Utilizar **colecciones**, estructuras de datos y manejo de archivos  
- Desarrollar aplicaciones simples de consola en C#
